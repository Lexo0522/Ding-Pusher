# Ding Pusher 插件发行说明

## 插件概述

Ding Pusher 是一款 WordPress 插件，用于自动检测新文章并通过钉钉机器人推送，包含去重机制和新用户提示功能。

## 功能特点

### 1. 钉钉机器人多维度配置
- **Webhook 地址配置**：支持钉钉机器人 Webhook 地址输入和格式校验
- **安全验证**：支持关键词、加签、IP 白名单三种安全方式
- **消息类型**：支持文本、链接、Markdown 三种消息类型
- **模板自定义**：支持自定义文章推送和用户提示模板
- **触发场景**：支持新文章发布、文章更新、自定义文章类型推送

### 2. 去重机制
- **多维度去重标识**：按 post_ID 和「文章标题 + 发布时间」双重去重
- **去重记录管理**：支持手动标记/取消标记已推送记录
- **去重记录清理**：可配置去重记录保留时间，自动清理过期记录

### 3. 新用户提示
- **双场景新用户提示**：新用户注册和插件首次激活都会发送提示
- **模板自定义**：支持自定义新用户提示模板
- **推送开关**：新用户提示单独开关，可选择是否开启

### 4. 推送容错与重试机制
- **失败自动重试**：推送失败后自动重试，重试间隔递增
- **失败记录与提醒**：记录失败原因，便于排查问题
- **手动重推**：支持单条/批量手动重推

### 5. WordPress 原生自动更新
- 支持 WordPress 后台自动检测更新
- 一键升级插件
- 详细的更新日志

## 安装方法

### 方法一：直接上传
1. 下载插件压缩包
2. 登录 WordPress 后台，进入「插件」→「安装插件」→「上传插件」
3. 选择下载的压缩包，点击「现在安装」
4. 安装完成后，点击「激活插件」

### 方法二：FTP 上传
1. 下载插件压缩包并解压
2. 使用 FTP 工具连接服务器，将解压后的文件夹上传到 `/wp-content/plugins/` 目录
3. 登录 WordPress 后台，进入「插件」页面，找到 Ding Pusher 插件，点击「激活」

## 配置步骤

1. **创建钉钉机器人**：
   - 打开钉钉群，点击群设置 → 智能群助手 → 添加机器人
   - 选择「自定义」机器人，填写机器人名称
   - 选择安全设置（关键词、加签或 IP 白名单）
   - 复制生成的 Webhook 地址

2. **配置插件**：
   - 登录 WordPress 后台，进入「Ding Pusher」→「设置」
   - 粘贴 Webhook 地址
   - 选择安全验证方式，填写相应配置
   - 选择消息类型，编辑推送模板
   - 开启需要的触发场景
   - 点击「保存更改」

3. **测试推送**：
   - 点击「发送测试消息」按钮
   - 检查钉钉群是否收到测试消息

## 使用说明

### 推送记录管理
- 进入「Ding Pusher」→「推送记录」页面
- 查看已推送的文章列表
- 可点击「取消标记」按钮，将文章标记为未推送状态
- 可点击「清理所有记录」按钮，清理所有推送记录

### 手动触发推送
- 对于未推送的文章，可重新发布文章触发推送
- 对于已推送的文章，可先取消标记，再重新发布触发推送

### 常见问题

#### Q: 推送失败怎么办？
A: 请检查以下几点：
- Webhook 地址是否正确
- 安全设置是否匹配（关键词、加签、IP 白名单）
- 网络连接是否正常
- 查看 WordPress 日志获取详细错误信息

#### Q: 如何重新推送已推送的文章？
A: 在「推送记录」页面，点击「取消标记」按钮，然后重新发布文章或等待定时任务检查。

#### Q: 插件支持哪些文章类型？
A: 插件默认支持文章（post）类型，可在设置中开启自定义文章类型支持。

#### Q: 如何修改推送频率？
A: 在「设置」页面的「高级设置」中，修改「推送间隔」即可。
